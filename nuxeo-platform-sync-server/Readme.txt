The server module providing synchronization feature. It stays on the online server.
The features provided:
- runs a client provided query and supplies the tuples describing the current situatio non server
- supplies the DocumentSnapshot at request for updating a certain document (being usual document, version or proxy)
- exports through restlet the vocabularies 
- exports through restlet the relations
- exports through restlet a document providing the blobs
